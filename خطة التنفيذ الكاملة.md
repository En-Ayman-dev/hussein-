# 🧱 المرحلة 1: Ingestion Pipeline (TTL → DB)

## 🎯 الهدف

تحويل ملف TTL إلى قاعدة بيانات قابلة للاستعلام

---

## 🔹 Task 1.1 — بناء TTL Parser

### Prompt جاهز:

```text
Build a Python module (ttl_parser.py) that:
- Loads a TTL file using rdflib
- Extracts:
  - Concepts (URI, rdfs:label, definition, quote, actions, importance)
  - Synonyms
  - Relations (causes, opposes, establishes, isMeansFor)
- Returns structured Python dict
- Handles malformed TTL safely
- Ignores unsupported RDF-star syntax
```

---

## 🔹 Task 1.2 — تنظيف البيانات

### Prompt:

```text
Create a normalization module that:
- Normalizes Arabic text
- Removes diacritics
- Normalizes Alef variants
- Lowercases text
- Generates normalized_label for search
```

---

## 🔹 Task 1.3 — تخزين البيانات

### Prompt:

```text
Create SQLAlchemy models for:
- concepts
- concept_synonyms
- concept_relations
- documents
Include pgvector embedding column
```

---

## 🔹 Task 1.4 — بناء Embeddings

### Prompt:

```text
Write a function that:
- Takes concept text (label + definition + quote)
- Generates embedding using OpenAI
- Stores vector in PostgreSQL pgvector column
- Handles batching and retry
```

---

## 🔹 Task 1.5 — Endpoint رفع TTL

### Prompt:

```text
Build FastAPI endpoint /api/ontology/upload that:
- Accepts .ttl file
- Validates size and type
- Parses file
- Stores extracted data in DB
- Returns number of concepts and relations
```

# 🧱 المرحلة 2: Query Engine

## 🎯 الهدف

تحليل السؤال ومطابقة المفاهيم

---

## 🔹 Task 2.1 — Query Analyzer

### Prompt:

```text
Build a module query_analyzer.py that:
- Takes Arabic question
- Classifies intent into:
  - definition
  - cause
  - solution
  - comparison
  - unknown
- Extracts keywords
Use rule-based approach first
Fallback to LLM if uncertain
```

---

## 🔹 Task 2.2 — Concept Matcher

### Prompt:

```text
Build concept_matcher.py that:
- Searches concepts using:
  1. exact label match
  2. synonym match
  3. PostgreSQL full-text search
  4. vector similarity (pgvector)
- Returns top 3 candidate concepts
- Includes confidence score
```

---

## 🔹 Task 2.3 — Re-ranking

### Prompt:

```text
Implement ranking logic:
- exact match = highest score
- synonym match = high
- importance="main" boost
- embedding similarity
Return best concept
```
